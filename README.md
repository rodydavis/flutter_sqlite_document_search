# flutter_sqlite_document_search

Local SQLite embeddings with [sqlite-vec](https://github.com/asg017/sqlite-vec).


## Getting Started

Run the following:

```bash
make deps
make build_files
make run_macos
```

Then add markdown and text files in the app to query against.
